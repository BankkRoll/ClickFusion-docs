## Coolmode Effect

### Introduction

The Coolmode effect comes with a variety of options to customize particles and their behavior.

### Properties

<Accordion title="Particle Options">

**Particle Options**

- `particle`: (optional) URL for custom particle image
- `size`: (optional) Size of the particle
- `speedHorz`: (optional) Horizontal speed of the particle
- `speedUp`: (optional) Upward speed of the particle

```tsx
// Code example using all props for Coolmode
<ClickFusion 
  effect="coolmode"
  particleOptions={{
    particle: 'customParticleImgURL',
    size: 40,
    speedHorz: 5,
    speedUp: 10
  }}
>
  <button>
    Click Me for Cool Effect!
  </button>
</ClickFusion>
```

</Accordion>

## Rainmode Effect

### Introduction

The Rainmode effect offers options to control the falling speed and the number of particles.

### Properties

<Accordion title="Particle Options">

**Particle Options**

- `particle`: (optional) URL for custom particle image
- `size`: (optional) Size of the particle
- `speedDown`: (optional) Downward speed of the particle
- `particleCount`: (optional) Number of particles

```tsx
// Code example using all props for Rainmode
<ClickFusion 
  effect="rainmode"
  particleOptions={{
    particle: 'customParticleImgURL',
    size: 50,
    speedDown: 5,
    particleCount: 30
  }}
>
  <button>
    Click Me for Rain Effect!
  </button>
</ClickFusion>
```

</Accordion>

## Code Mode Effect

### Introduction

The Code Mode effect offers options to control the falling code-like particles and their behavior.

### Properties

<Accordion title="Particle Options">

**Particle Options**

- `particleCount`: (optional) Number of particles
- `speedDown`: (optional) Downward speed of the particles
- `size`: (optional) Size of the particles
- `color`: (optional) Color of the particles "light" | "dark"

```tsx
// Code example using all props for Code Mode
<ClickFusion 
  effect="codemode"
  particleOptions={{
    particleCount: 40,
    speedDown: 5,
    size: 10,
    color: 'light'
  }}
>
  <button>
    Click Me for Code Falling Effect!
  </button>
</ClickFusion>
```

</Accordion>

## Confetti Mode Effect

### Introduction

The Confetti Mode effect allows you to set the number of confetti particles, their falling speed, and color modes.

### Properties

<Accordion title="Particle Options">

**Particle Options**

- `particleCount`: (optional) Number of particles
- `speedDown`: (optional) Downward speed of the particles
- `size`: (optional) Size of the particles
- `color`: (optional) Color mode of the particles "red", "orange", "yellow", "green", "blue", "purple"

```tsx
// Code example using all props for Confetti Mode
<ClickFusion 
  effect="confettimode"
  particleOptions={{
    particleCount: 100,
    speedDown: 5,
    size: 20,
    color: 'purple'
  }}
>
  <button>
    Click Me for Confetti Effect!
  </button>
</ClickFusion>
```

</Accordion>
